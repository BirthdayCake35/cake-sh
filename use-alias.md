# Use alias on cake.sh

use of alias

# macOS

first open terminal and type this code

```bash
open ~/.bash_profile
```

Go to the bottom of the file and write this code

```bash
alias cake="path/your/cake.sh"
```

# Linux

first open terminal and type this code

```bash
open ~/.bashrc
```

Go to the bottom of the file and write this code

```bash
alias cake="path/your/cake.sh"
```
